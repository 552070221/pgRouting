pgr_version¶

Nom¶

pgr_version — Requête pour l’information de version de pgRouting.

Synopsis¶

Retourne un tableau avec l’information de version pgRouting.

table() pgr_version();

Description¶

Retourne un tableau avec :

version:varchar version de pgRouting
tag:varchar étiquette Git du build pgRouting
hash:varchar hash Git du build
branch:varchar branche Git du build pgRouting
boost:varchar version Boost

Histoire

  • Nouveau depuis la version 2.0.0

Exemples¶

  • Requête pour la version string complète
SELECT pgr_version();

                     pgr_version
------------------------------------------------------
 (2.0.0-dev,v2.0dev,290,c64bcb9,sew-devel-2_0,1.49.0)
(1 row)
  • Requête pour les attributs version et boost
SELECT version, boost FROM pgr_version();

  version  | boost
-----------+--------
 2.0.0-dev | 1.49.0
(1 row)

Voir aussi¶

  • pgr_versionless to compare two version numbers
