Use’s Recipes contributions¶

Cómo contribuir.

Utilice un issue tracker en git (ver Soporte) con un título que contiene: Proposing a Recipe: MiNombreDeReceta. El cuerpo debe contiener:

  • Autor: requerido
  • Correo: Si usted está suscrito a la lista de los desarrolladores no es necesario
  • Fecha: fecha
  • Comentarios y código: formato reStructuredText

Cualquier contacto con el autor se hará utilizando la lista de correo de desarrolladores. El equipo pgRouting evaluará la receta y será incluido en esta sección cuando sea aprobada.

Comparando la topología de una red de que no tiene nodos en las intersecciones con una red con nodos en las intersecciones¶

Author:Equipo pgRouting.

Esta receta se utiliza la red de ejemplo.

SELECT pgr_createTopology('edge_table', 0.001);
SELECT pgr_analyzegraph('edge_table', 0.001);
SELECT pgr_nodeNetwork('edge_table', 0.001);
SELECT pgr_createTopology('edge_table_noded', 0.001);
SELECT pgr_analyzegraph('edge_table_noded', 0.001);
