Contribuciones de envolturas de los usuarios¶

Cómo contribuir.

Utilice un issue tracker en git (ver Soporte) con un título que contiene: Proposing a wrapper: MiNombreDeENVOLTORIO. El cuerpo debe contener:

  • Autor: requerido
  • Correo: Si usted está suscrito a la lista de los desarrolladores no es necesario
  • Fecha: fecha
  • Comentarios y código: formato reStructuredText

Cualquier contacto con el autor se hará utilizando la lista de correo de desarrolladores. El equipo pgRouting evaluará el envoltorio y será incluido en esta sección cuando sea aprobado.
