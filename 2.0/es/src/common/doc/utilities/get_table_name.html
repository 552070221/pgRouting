pgr_getTableName¶

Nombre¶

pgr_getTableName — recupera el nombre de la columna tal cual se almacena en las tablas de la administración de postgres.

Nota

Esta función es para ayudar a los desarrolladores

Sinopsis¶

Devuelve un registro que contiene los nombres registrados de la tabla y del esquema al que pertenece.

(text sname, text tname)  pgr_getTableName(text tab)

Descripción¶

Parámetros

tab:text nombre de la tabla con o sin componente de esquema

Regresa

sname:
  • text que contiene el nombre registrado del esquema de la tabla “tab”.
    • cuando el esquema no proporciono en “tab”, se usa el esquema actual.
  • NULL cuando:
    • El esquema no se encuentróen las tablas de la administración de postgres.
tname:
  • text que contiene el nombre registrado de la tabla “tab”.
  • NULL cuando:
    • El esquema no se encuentróen las tablas de la administración de postgres.
    • La tabla “tab” no está registrada bajo el esquema de sname en las tablas de la administración de postgres

Historia

  • Nuevo en la versión 2.0.0

Ejemplos¶

SELECT * from pgr_getTableName('edge_table');

 sname  |   tname
--------+------------
 public | edge_table
(1 row)

SELECT * from pgr_getTableName('EdgeTable');

 sname  |   tname
--------+------------
 public |
(1 row)

SELECT * from pgr_getTableName('data.Edge_Table');
 sname | tname
-------+-------
       |
(1 row)

Los ejemplos utilizan el: ref: ‘sampledata’.
